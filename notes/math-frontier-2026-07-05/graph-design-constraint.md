# Pre-computed constraint: harmonic receipts need a new pair design

Computed 2026-07-05 (exact rational rank, script in session log): the JCB
pair design (8 nodes, strides {1,2,4}, 20 edges) has cycle dimension 13
and 16 triangles whose boundary space has rank exactly 13.

**Harmonic dimension = 0.** Any Hodge "harmonic component" measured on
this graph is identically zero by construction — the full cyclic residual
is expressible in triangle curls, so curl-vs-harmonic decomposition
carries no information HERE regardless of the judge.

Design consequence: to measure globally-cyclic-but-triangle-invisible
judgment structure (the harmonic part), the comparison graph must contain
chordless cycles of length ≥ 4 that triangles cannot span. Example: a
stride-{1,3} ring on 8 nodes has ZERO triangles (its whole 9-dim cycle
space is harmonic); a mixed design — one triangle-rich block plus one
chordless-cycle block — separates local from global cyclicity in a single
run. The receipt for any future harmonic instrument must state its
graph's harmonic dimension the way rates state denominators.
